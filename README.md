# MedGrid

MedGrid is a static, client-side PDF N-up converter for study notes and lecture slides.

- Single-file app: `index.html`
- Works entirely in the browser
- Uses PDF.js for preview and pdf-lib for original-quality PDF embedding
- Supports folders, notes, presets, live preview, and Turkish/English UI

## Deploy

This project can be deployed as a static site with GitHub Pages or Vercel. No build step is required.
